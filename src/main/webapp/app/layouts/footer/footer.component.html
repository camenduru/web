Tost is a free, non-profit, open-source service that serves inference for the latest AI papers using the Non Profit GPU Cluster. Tost does not store any inference data; all data expires within 12 hours. However, Tost offers the option to send data to your Discord channel. Tost provides 100 free wallet balance to each account every day. If you want 1100 wallet balance a day ($22/month), please subscribe to GitHub Sponsors or Patreon. Tost sends all profits from demos to the paper's first author. Tost's budget is supported by companies and individual sponsors.
